# Cardano tokens and staking application releases:

A desktop application for managing Cardano assets using Fireblocks custody. Built on Electron, it gives teams a self-contained interface to manage Cardano vault accounts, execute ADA and native token transfers through the Fireblocks signing infrastructure, handle the full Cardano staking lifecycle, and participate in on-chain governance.
